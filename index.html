Featured Projects

Multiplayer Elimination Game Online

Welcome to the captivating realm of Last Card Standing, a testament to the synergy between innovative programming and cutting-edge technology. Crafted with precision using a robust tech stack, Last Card Standing embodies excellence in game development. Leveraging HTML, CSS, JavaScript, Socket.IO, Express.js, Node.js, and MySQL. This project stands at the forefront of modern web-based gaming experiences, having boasted over 100 active players in a day and an impressive user engagement rate exceeding 80%.

At its core, Last Card Standing harnesses the power of JavaScript and Socket.IO to facilitate real-time communication between players, creating a dynamic and immersive multiplayer environment. Express.js, a minimalist web framework for Node.js, forms the backbone of the server-side logic, enabling seamless integration of features such as lobby creation, player joining, and game management. The MySQL database ensures efficient data storage and retrieval, providing a reliable foundation for persisting game state, player information, and lobby details.

3D Adventure Game (Castle Climbers)

Castle Climber 3D, a range of devices and tools were utilized to create an immersive gaming experience. The project primarily relied on a high-performance desktop workstation equipped with powerful hardware components, including a multi-core processor, dedicated graphics card, and ample RAM, to handle the demands of 3D rendering, physics simulations, and real-time gameplay. Overall, the combination of powerful hardware, versatile software tools, and effective project management techniques contributed to the successful creation of Castle Climber 3D and its immersive gameplay experience.
